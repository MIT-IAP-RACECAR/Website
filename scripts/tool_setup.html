Field Description Examples Input Errors
Absolute path The full path to the directory on your computer in which you want to store your RACECAR files Windows: C:\Users\sally\Documents
Mac/Linux: /home/sally/documents
IP address The IP address of your RACECAR, which should be provided by your instructor. 192.168.1.200
Team name The name of your team, which must contain only characters and numbers sally
Operating system The operating system of your computer Windows, Mac, Linux


Troubleshooting:

1. If you are using Windows, you must use bash, not command prompt. You can install bash by downloading Ubuntu from the Windows store: https://docs.microsoft.com/en-us/windows/wsl/install-win10
